# Sistema Bancário em Java

## Aluna
Nicolly Ivinna

## Turma
2 DS 1/P

## Descrição

Projeto desenvolvido em Java utilizando os conceitos de Programação Orientada a Objetos (POO). O sistema permite realizar operações bancárias básicas por meio de um menu interativo no terminal.

## Funcionalidades implementadas

- Cadastro de contas
- Conta Corrente
- Conta Poupança
- Depósito
- Saque
- Transferência
- Consulta de saldo
- Listagem de contas
- Encapsulamento
- Herança
- Polimorfismo

## Como executar

1. Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou NetBeans).
2. Compile os arquivos.
3. Execute a classe Main.
4. Utilize o menu para realizar as operações bancárias.
